14 // Описание алгоритма.
15 // Массив состоит из двух смежных (последовательно расположенных в памяти) буфферов,
16 // каждый из которых имеет размер окна. Новые данные пишутся в каждый буффер, то есть записываются дважды.
17 // Окно скользит по пространству обоих буфферов, постепенно переползая из первого буффера во второй.
18 // По достижению конца второго буфера, окно смещается к началу первого (в позицию begin+1).
19 // При этом, данные в окне выглядят так, как будто оно продолжает ползти по данным последовательно.
